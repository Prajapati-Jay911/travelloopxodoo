# Traveloop – Complete UI Mockup Description

The provided wireframes represent a travel planning and trip management application called **Traveloop**.
The application includes authentication, trip planning, itinerary management, community interaction, trip journaling, packing management, and admin analytics.

---

# 1. Login Screen (Screen 1)

## Purpose

The login screen allows existing users to securely access the Traveloop platform.

## Main Components

* **Application Header**

  * Displays the app name “Traveloop”.

* **Logo/Profile Placeholder**

  * Circular image placeholder centered near the top.
  * Represents app logo or user avatar.

* **Username Input Field**

  * Text field for entering username/email.

* **Password Input Field**

  * Secure password entry field.

* **Login Button**

  * CTA button to authenticate the user.

## UX Goal

A simple and minimal authentication interface focused on quick login access.
